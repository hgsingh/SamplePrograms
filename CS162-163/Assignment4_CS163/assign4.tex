\begin{homeworkProblem}
 Ethical behavior in social media is comprised of using another persons (to which one is connected through a social network) information whether it be personal, or professional in ethical manner. For example if a person uses the D2L webpage and posts code for a certain assignment, it would be anothers responisibility to not use the code or variants of in their own assignment. However it is also a responibility of a person posting the material to decide whether it is ethical. The issue culminates to whether or not certain types of sensitive information can be shared in a network setting and really depends on what type of damage that information can do, if harmful, one should avoid sharing harmful information or misinformation. In the example of forwarding someones email, if the email is publically available then one can share it be association, however distributing others personal information in a network where it was not available to begin with, is unethical.
\end{homeworkProblem}
