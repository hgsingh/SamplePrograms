An outline for producing the multiplication table as shown in Malik Ch. 5 is, 

\begin{enumerate}
 \item Set up a iterator that iterates through the first 5 natural numbers, e.g. for i in range 1 to 5.
    \begin{enumerate}
    \item Set another iterator nesting it inside the first iterator. This loops a set number of times for each iteration defined in its scope. e.g. for i in range 1 to 10. 
    \item Multiply every iteration by the iteration defined in the scope.
    \item Display the number using the output buffer.
    \end{enumerate}
 \item Purge output buffer here, e.g. endl;.
\end{enumerate}

\end{homeworkProblem}
